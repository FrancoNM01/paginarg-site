# Paginarg Store Theme

Tema base para WordPress + WooCommerce inspirado en la demo ecommerce de PAGINARG.

## Archivos principales

- `style.css`: estilos base del tema
- `functions.php`: soporte WooCommerce, helpers y shell visual de tienda
- `header.php` y `footer.php`: cabecera y pie con estructura tipo demo
- `front-page.php`: home comercial inspirada en la demo estatica
- `woocommerce.php`: envoltorio para paginas WooCommerce

## Como instalar

1. Copia la carpeta `paginarg-store` dentro de `wp-content/themes/`.
2. En WordPress, activa el tema desde `Appearance > Themes`.
3. Instala y activa WooCommerce.
4. Crea o revisa estas categorias de producto:
   - `women`
   - `men`
   - `accessories`
   - `new-arrivals`
5. Asigna una pagina como home estatica para aprovechar `front-page.php`.
6. Crea las paginas de WooCommerce: Shop, Cart, Checkout y My Account.
7. Ajusta logo, moneda, textos y colores si queres llevarlo mas hacia la marca final.

## Recomendacion de uso

Este tema es una base visual. Lo ideal es usarlo como punto de partida y despues adaptar:

- logo real
- fotos reales de producto
- menu final
- pagina de contacto
- checkout y textos legales
